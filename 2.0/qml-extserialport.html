    ExtSerialPort

    Enables you to connect and communicate with a serial port. More...

    Detailed Description

        import QtExtSerialPort 2.0

    ExtSerialPort
    {
        id: port
        portName: "COM22"
        baudRate: 19200
        parity: ExtSerialPort.PAR_EVEN
        dataBits: ExtSerialPort.DATA_6
        connected: true
        //shown data in TextEdit
        onDataAvailable: recvText.text = port.stringData
    }

    See also QextSerialPort.

    Property Documentation

    baudRate : int

    Gets and sets the baudRate associated with the serial port


    connected : bool

    This property holds the device state of the port. If the port is opened, it returns true. It can be set true or false to control the port.


    dataBits : ExtSerialPort::DataBitsType

    Gets and sets the dataBits associated with the serial port

    • ExtSerialPort.DATA_8
    • ExtSerialPort.DATA_7
    • ExtSerialPort.DATA_6
    • ExtSerialPort.DATA_5

    flowControl : ExtSerialPort::FlowType

    Gets and sets the flowControl associated with the serial port

    • ExtSerialPort.FLOW_OFF - No flow control (default)
    • ExtSerialPort.FLOW_HARDWARE - Hardware (RTS/CTS) flow control
    • ExtSerialPort.FLOW_XONXOFF - Software (XON/XOFF) flow control

    parity : ExtSerialPort::ParityType

    Gets and sets the parity associated with the serial port

    • ExtSerialPort.PAR_NONE - (default)
    • ExtSerialPort.PAR_ODD
    • ExtSerialPort.PAR_EVEN
    • ExtSerialPort.PAR_MARK - Windows only
    • ExtSerialPort.PAR_SPACE

    portName : string

    Gets and sets the port name associated with the serial port


    stopBits : ExtSerialPort::StopBitsType

    Gets and sets the stopBits associated with the serial port

    • ExtSerialPort.STOP_1 - (default)
    • ExtSerialPort.STOP_1_5 - Windows only
    • ExtSerialPort.STOP_2

    stringCodec : string

    This property controls which QTextCodec is used to convert QByteArray to and from QString.


    stringData : string

    This property receives or sends data to serial port device. Arrival of data is signaled by the dataAvailable signal and can be read by stringData. Calling sendStringData will transmit the string.
