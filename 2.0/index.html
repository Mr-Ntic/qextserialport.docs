Qt ExtSerialPort

It currently supports Mac OS X, Windows, Linux, FreeBSD.

Getting Started

Qt ExtSerialPort provides classes for controlling serial ports. To include the definitions of the module's classes, use the following directive:

#include <QtExtSerialPort>

To link against the module, add this line to your qmake .pro file:

QT += extserialport
