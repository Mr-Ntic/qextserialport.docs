    Detailed Description

    The QextSerialHelper class provides some convenient functions to ease the use of QextSerialPort.

    Member Function Documentation

    QString QextSerialHelper::fromBaudRateType(QextSerialPort::BaudRateType value, bool * ok = 0) [static]

    Returns the string representation of QextSerialPort::BaudRateType value. Returns a default string "BAUD9600" if the conversion fails, for example, (QextSerialPort::BaudRateType)9999 is passed as parameter.

    If a conversion error occurs, * ok is set to false; otherwise * ok is set to true.

    See also toBaudRateType().
